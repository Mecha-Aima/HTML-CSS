Hello world

Let's dive right in. This program first prompts the user for their name and then stores it in the name variable. Finally, it prints a greeting message personalized with the user's name.

name = input("What's your name? ")
print("Hello,", name + "!")

The print() function is used to display information to the output console. The input() function is used to enter data from the keyboard.

Data types

In Python, you can create variables and assign values to them. Python supports various data types including integers, floats, strings, lists, tuples, and dictionaries.

name = "Alice" # String data type
age = 30 # Integer data type
is_happy = True # Boolean data type (True or False)

We can then use these variables in our code to perform operations or display information.

Operators

Python provides various operators for performing operations on variables and values. These include arithmetic operators, comparison operators, logical operators, and more.

# Arithmetic operators
a = 10
b = 5
print(a + b)  # Addition
print(a - b)  # Subtraction
print(a * b)  # Multiplication
print(a / b)  # Division

# Comparison operators
print(a == b) # Equal to
print(a != b) # Not equal to
print(a > b)  # Greater than
print(a <= b) # Less than or equal to

# Logical operators
x = True
y = False
print(x and y) # Logical AND
print(x or y)  # Logical OR
print(not x)   # Logical NOT

These operators have their use in a number of mathematical applications that require data processing.

Control flow

Python provides control structures such as if statements, loops, and functions to control the flow of your program.

age = 15
if age >= 18:
  print("You are eligible to vote.")
else:
  print("You are not eligible to vote.")

This code checks the value of age and displays a message based on the condition.

Functions

Functions in Python allow you to encapsulate reusable pieces of code. You can define your own functions using the def keyword.

def greet(name):
    print("Hello, " + name + "!")

greet("Alice")

This code defines a function named greet() that takes in a single arguement name
